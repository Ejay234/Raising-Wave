What to know

Raising Waves is a machine learning model that predicts the year of the sea level based on two factors:
    Sea level rise is caused primarily by two factors related to global warming: the added water from melting ice sheets and glaciers and the expansion of seawater as it warms.*Earthdata Nasa*

I used python as my main programming language to create the ML model using the sklearn library. The ML model is a supervised linear regression model that takes 4 features to predict the year.

Why this project

This project is created with the purpose to spread awareness of the rising sea levels over the course of the past years.

The intended audience of this project is meant for those who will be affecting by the flooding. Like myself and my community, we live on a small island in the Pacific Ocean. Due to that fact, I created this ML model to educate my peers and policymakers on island, to start implementing procedures/policies that can slow down global warming.
